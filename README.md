# NovaCore Knowledge AI

Enterprise Knowledge Assistant powered by Retrieval-Augmented Generation (RAG).

## Overview

NovaCore Knowledge AI is an enterprise AI assistant capable of answering questions using internal corporate documentation.

The application indexes company documents, creates vector embeddings, and retrieves relevant information to generate grounded responses using a Large Language Model (LLM).

## Features

- PDF document ingestion
- CSV document ingestion
- Vector search with ChromaDB
- Retrieval-Augmented Generation (RAG)
- Streamlit interface
- Modular architecture
- Oracle Cloud deployment
